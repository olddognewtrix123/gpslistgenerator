Welcome to the GPS List Generator.

This application accepts a semicolon delimited string of street addresses and returns a

corresponding semicolon-delimited list of GPS locations.

The GPS List Generator produces only the raw data on the screen and no formatted data export.

The fact that the list is semicolon-delimited should enable formatting in Excel or a similar application

To use the GPS List Generator, first you must obtain your own Google Maps API key.

Please note that since this is a front-end application only (it is just a bunch of code that runs

in the browser) the GPS List Generator is by design NOT set up to record any information that gets

plugged into it. The only way there could be a database running in the background recording what

you type is if your internet provider is doing something nefarious. GPS List Generator itself does

not hang on to anyone's information.

Have your list of addresses that you want the matching GPS locations ready and formatted like so:

"7301 Buttercup Rd,Sykesville Maryland;7555 Waterloo Rd,Jessup Maryland;30420 Revells Neck Rd,Westover Maryland"
